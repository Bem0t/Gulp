Для нового проекта:
1)Запуск git-bash в корне проекта
2)npm i

--------------------------------------------------- Если что-то отказывается работать
npm i -D webp-converter@2.2.3
npm i -D webpack
---------------------------------------------------
npm run dev (Режим разработчика)
npm run build (Режим продакшена) 
npm run zip (Создать архив)
npm run svgSprive 
npm run deploy (Загружает файлы на сервер)
------------------------------------------------------------
HELP =>
1)Если выдает оошибку компиляции в html модуль
	1) проверить, есть ли в названии картинки пробелы
	2) идти плакать
2)Если многостраничный сайт, то просто создаем копию index.html и переименовывем под нужную страницу
3) Для обновления файла шрифтов, нужно удалить fonts.scss
4)Папка которая создается на сервере
    4.1) название меняется в config/path.js (ftp: "")
    4.2) Параметры для подключения задаются в config/ftp.js
5)Для нормального подключения картинок, нужно расширение Path Autocomplete 
	5.1)ctrl + shift + p
	5.2)Вставляем код 
	"path-autocomplete.pathMappings": {
        "@img": "${folder}/src/img",
        "@scss": "${folder}/src/scss",
        "@js": "${folder}/src/js",
    },
    	5.3) Пути указываем src="@img/"
